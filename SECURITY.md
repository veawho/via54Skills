# 安全政策 (Security Policy)

## 中文

### 报告漏洞

**请勿公开 issue** 报告安全问题.

**私下发邮件给**: veawho (巫师叔叔) via https://github.com/veawho

请提供: 漏洞描述, 复现步骤, 影响范围, 修复建议.

### 支持的版本

| 版本 | 支持 |
|------|------|
| 最新 | ✅ |
| < 最新 | ❌ |

### 流程

1. 收到漏洞报告 (私下)
2. 维护者确认
3. 准备修复
4. 私下发安全补丁
5. 公开 release notes

### Reporting

**DO NOT** open public issues for security vulnerabilities. Email privately to: veawho via https://github.com/veawho

Include: vulnerability description, reproduction steps, impact, fix suggestions.
