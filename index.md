A curated collection of personal AI agent skills, all following the `via54<action>` naming convention:

## Naming Convention

All skills named `via54<lowercase-action>`:
- `via54larkgroups` ← LARK + groups
- `via54design` ← DESIGN
- `via54github` ← GitHub
- `via54medit` ← MED + IT (medical literature)
- `via54idea` ← IDEA
- `via54larkfix` ← LARK + fix
- `via54hermes` ← HERMES
- `via54larkbotgo` ← LARK + bot + go
